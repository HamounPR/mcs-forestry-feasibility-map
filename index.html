About this map (README)
Scope Megawatt charging station (MCS) feasibility for forest-to-sawmill transport links across 37 sawmills.
How to use Select a detour threshold (1 / 5 / 10 km) and then choose a sawmill to visualize feasible ODs.
Method For each forest–sawmill OD pair, routing is performed on the national public road network. Each OD is assigned the nearest MCS based on minimum additional travel distance.
Computed per OD:
  • Direct route length (km)
  • Forest → MCS distance
  • MCS → Sawmill distance
  • Total route via MCS and detour (absolute & relative)
Detour-based feasibility ODs are considered feasible if the MCS-based route adds less than the selected detour threshold (<1 km, <5 km, or <10 km), under the base constraint of route <150 km.
Fallback mechanism If no OD satisfies the base distance constraint, the allowable route length is expanded 150 → 200 → 250 → 300 → 350 km, and the OD with minimum detour is selected. This highlights structural infrastructure gaps in sparse regions.
Tip: You can click sawmill markers to explore the site context, then use the dropdown to display ODs.
